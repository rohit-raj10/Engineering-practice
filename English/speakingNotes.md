# 📘 Vocabulary Handbook

## Engineering Words

### 1. Latency Budget

**Meaning:** The maximum acceptable response time allocated to each component in a request path.

**Example:**
> We exceeded our latency budget after adding an extra database query.

---

### 2. Bottleneck *(Review)*

**Meaning:** The slowest component limiting overall system performance.

**Example:**
> Database writes became the bottleneck.

---

### 3. Idempotency

**Meaning:** Repeating an operation produces the same result.

**Example:**
> Payment APIs should support idempotency.

---

### 4. Throughput *(Review)*

**Meaning:** The amount of work completed per unit of time.

---

### 5. Observability *(Review)*

**Meaning:** Understanding a system through logs, metrics, and traces.
